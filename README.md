# 📢 층간소음/코골이 감지기 (Noise Logger)

설치 없이 웹 브라우저에서 바로 실행하는 소음 감지 및 로그 기록 도구입니다.
스마트폰이나 PC의 마이크를 통해 주변 소음을 측정하고, 설정된 기준값(dB)을 초과하는 순간을 자동으로 기록합니다.

> **[데모 페이지 바로가기](https://여기에_네_깃허브_페이지_주소를_넣으세요)**
> 
> *(예: https://your-id.github.io/noise-logger)*

## 🎯 주요 기능
- **실시간 데시벨 측정:** 웹 오디오 API를 활용한 실시간 소음 모니터링
- **임계값(Threshold) 설정:** 사용자가 직접 감지 기준(dB) 조절 가능
- **자동 로그 기록:** 기준치를 넘는 소음 발생 시 시각과 데시벨 자동 저장
- **프라이버시 보호:** 별도 서버로 데이터를 전송하지 않고, 브라우저 내에서만 작동 (새로고침 시 초기화)

## 📝 사용 방법
1. 위 링크를 통해 웹사이트에 접속합니다. (Chrome, Safari 권장)
2. **'감지 시작'** 버튼을 누르고 마이크 권한을 **[허용]**합니다.
3. 슬라이더를 움직여 감지할 **소음 기준값(Threshold)**을 설정합니다.
   - *평소 조용한 상태에서는 초록색이 유지되도록, 감지하고 싶은 소음 크기보다는 조금 낮게 설정하세요.*
4. 소음이 발생하면 게이지가 **빨간색**으로 변하며 아래 로그창에 기록됩니다.

## ⚠️ 주의사항 (필독)
이 앱은 웹 기반 프로토타입이므로 백그라운드 실행을 완벽하게 지원하지 않습니다.

1. **화면 유지:** 장시간(수면 중 등) 측정 시 **스마트폰 화면이 꺼지지 않도록 설정**하고, 충전기를 연결해주세요.
   - 화면이 꺼지거나 브라우저가 백그라운드로 넘어가면 OS 정책상 마이크가 차단될 수 있습니다.
2. **데이터 저장:** 페이지를 **새로고침(F5)하면 기록된 로그가 사라집니다.** 필요한 기록은 스크린샷이나 복사로 미리 저장하세요.

## 🛠 기술 스택
- HTML5 / CSS3
- Vanilla JavaScript
- Web Audio API
